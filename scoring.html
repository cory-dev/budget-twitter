Scoring Guide

Our subjective guidelines towards scoring your tweets

  • Misinformation:
    1. If more than 5 users have rated the tweet as misinfo, an option to hide the tweet is presented.
    2. Each user can vote only once.

  • Political Bias:
    1. There are 4 options available.
    2. Left bias wtih Democrats logo.
    3. Right bias with Republican logo.
    4. Centrists with Centrist logo.
    5. Bias unknown with a Question mark.
    6. Each user can vote only once.
